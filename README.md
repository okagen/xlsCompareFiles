# xlsCompareFiles
Compare CSV files or Excel files, then find differences. / CSVファイル、またはExcelファイルを突合して異なる個所を見つける。

## Sample direcotry structure.

~~~
root directory
│  xlsCompareFiles.xlsm
│
└─Data
    ├─Comp_A
    │      H19.csv
    │      H20.csv
    │      H21+noise.csv
    │
    └─Comp_B
            H19.csv
            H21.csv
~~~
